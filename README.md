# Pedestrian-Safety-System

> 라이다(LiDAR) 센서 기반 시각장애인 보행 안전 보조 시스템 — 공중 장애물을 감지하고 방향별 청각 경고음으로 실시간 위험을 알립니다.

---

##  프로젝트 개요

시각장애인은 흰 지팡이로 지면의 장애물은 인식할 수 있지만, **공중에 돌출된 간판·가지·구조물** 등은 감지할 수 없습니다.  
본 시스템은 이 사각지대를 해소하기 위해 LiDAR 센서로 주변 장애물의 위치와 거리를 실시간 측정하고,  
**이어폰을 통한 방향별 경고음**으로 사용자가 충돌 전에 회피할 수 있도록 지원합니다.

---

##  주요 기능

| 기능 | 설명 |
|------|------|
| 공중 장애물 감지 | LiDAR 센서로 지면 위 공간의 장애물 실시간 탐지 |
| 방향 구분 경고 | 장애물이 왼쪽·오른쪽 어느 방향인지 각각의 이어폰 채널로 구분 전달 |
| 거리 비례 BPM | 장애물이 가까울수록 경고음 BPM(박자)이 상승하여 긴박감 증폭 |
| 실시간 처리 | 센서 데이터 수신부터 경고음 출력까지 지연 최소화 |

---

##  경고 방식

```
장애물 방향  →  해당 방향 이어폰 채널 활성화
거리 (far)   →  낮은 BPM (여유 있는 경고음)
거리 (near)  →  높은 BPM (빠르고 긴박한 경고음)
```

- 왼쪽 장애물 → 왼쪽 이어폰에서 경고음 출력  
- 오른쪽 장애물 → 오른쪽 이어폰에서 경고음 출력  
- 거리가 가까워질수록 BPM 비례 증가 → 직관적 위험도 전달

---

##  기술 스택

- **센서**: LiDAR (라이다 거리 측정 센서)
- **언어**: Python
- **오디오 출력**: 스테레오 이어폰 (좌/우 채널 독립 제어)
- **플랫폼**: 임베디드 / 엣지 디바이스 (휴대용 구성)

---

##  시스템 구조

```
[LiDAR 센서]
     ↓  거리 + 방향 데이터
[장애물 분석 모듈]
     ↓  방향 판별 (좌/우) + 거리 계산
[BPM 생성 모듈]
     ↓  거리 → BPM 매핑
[스테레오 오디오 출력]
     ↓
[좌/우 이어폰 경고음]
```

##  개발 동기

> "시각장애인은 흰 지팡이로 지면을 파악할 수 있지만, 공중에 뻗은 간판이나 나뭇가지 같은 장애물은 전혀 알 수 없다."

이 문제를 해결하기 위해, 별도의 학습 없이 누구나 직관적으로 사용할 수 있는 **청각 피드백 기반 보행 보조 시스템**을 설계하였습니다.
