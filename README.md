# Pedestrian-Safety-System

> 딥러닝 기반 보행자 안전 모니터링 시스템 — 객체 탐지와 위험도 분석을 통해 야간 및 취약 구역의 보행 안전성을 평가합니다.

---

## 📌 프로젝트 개요

본 프로젝트는 CCTV 영상 및 센서 데이터를 기반으로 **보행자 안전 위협 요소를 실시간으로 탐지·분석**하는 딥러닝 파이프라인입니다.  
객체 탐지 모델을 활용하여 보행자, 차량, 장애물 등을 인식하고, 위험도 분석 알고리즘을 통해 구역별 안전 수준을 정량화합니다.

---

## 🌟 주요 기능

| 기능 | 설명 |
|------|------|
| 보행자 객체 탐지 | 딥러닝 기반 실시간 보행자·차량 탐지 (YOLO 계열 모델) |
| 위험도 분석 | 탐지 결과 기반 구역별 위험 수준 정량화 및 분류 |
| 야간 안전 동선 분석 | 조도 및 보행자 밀도 정보를 활용한 안전 동선 도출 |
| 핫스팟 시각화 | 위험 구역 히트맵 및 공간 분포 시각화 |
| 이력 데이터 관리 | 시계열 위험도 변화 추적 및 통계 분석 |

---

## 🛠️ 기술 스택

- **딥러닝 프레임워크**: Python, PyTorch / TensorFlow
- **객체 탐지 모델**: YOLO 계열 (YOLOv8 등)
- **데이터 처리**: OpenCV, NumPy, Pandas
- **시각화**: Matplotlib, Seaborn
- **플랫폼**: GPU 가속 환경 (CUDA)

---
